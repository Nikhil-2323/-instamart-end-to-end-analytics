🛒 Instamart End-to-End Analytics
📌 Project Overview

This project is an end-to-end retail analytics pipeline built using Python, MySQL, and Power BI.
The goal is to analyze sales data, track business performance, and generate actionable insights through interactive dashboards.

The project covers the complete data workflow:

Data Cleaning & Preprocessing (Python)

Data Storage & Querying (MySQL)

Data Visualization & KPI Reporting (Power BI)

🛠️ Tech Stack

Python – Pandas, NumPy, Matplotlib (Data Cleaning & Analysis)

MySQL – Data Storage & SQL Querying

Power BI – Dashboard Creation & KPI Visualization

📊 Key Features

Sales, Revenue & Profit Analysis

City-wise and Category-wise Performance

Time-based Order Analysis (Hourly / Monthly trends)

KPI Metrics (Total Orders, Total Revenue, Average Order Value)

Interactive Filters & Slicers

🗂️ Project Workflow

1️⃣ Data Preprocessing (Python)

Handled missing values

Data formatting & transformation

Feature engineering (Time slicing, Profit calculation)

2️⃣ Database Management (MySQL)

Created relational tables

Performed joins & aggregations

Wrote optimized SQL queries for analysis

3️⃣ Dashboard & Reporting (Power BI)

Built star schema data model

Created DAX measures

Designed interactive dashboard

📈 Sample Insights

Identified top-performing cities and categories

Analyzed peak order time slots

Evaluated profit margins across products

Discovered seasonal sales trends

🎯 Project Objective

To simulate a real-world retail business intelligence system and demonstrate strong skills in:

Data Cleaning

SQL Querying

Data Modeling

Business Analytics

Dashboard Design
